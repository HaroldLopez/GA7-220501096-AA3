Instrucciones de uso

1. Instalar Node.js.
2. Crear la base de datos ejecutando database.sql.
3. Abrir una terminal en la carpeta del proyecto.
4. Ejecutar:
   npm install
5. Ejecutar:
   npm start
6. Abrir:
   http://localhost:3000
